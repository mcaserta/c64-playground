# The Matrix Screensaver

## Available keys

| key | use |
|-----|-----|
| `B` | binary display mode |
| `X` | hex display mode |
| `G` | dna display mode |
| `R` | random (full charset) display mode (default) |
| `S` | slow speed mode |
| `M` | middle speed mode |
| `F` | fast speed mode |
| `N` | increments border color |
| `M` | increments background color |
| `Q` | quit |
